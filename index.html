Aviation Communications Assistant

Get a parsed ATIS transcript

This tool takes raw ATIS-style aviation communication text and converts it into a pilot-readable briefing card plus structured JSON. It uses deterministic parsing rules rather than AI, so the goal is not full natural-language understanding, but reliable extraction of key aviation fields from predictable transcript patterns.

Known Limitations

        
    
            
  • This parser expects predictable ATIS-style transcript patterns.
    • 
            
  • This does not currently handle “winds calm.”
    • 
            
  • Complex runway phrasing may overcapture.
    • 
            
  • Observation time parsing expects a simple “observation 1150 zulu” pattern.
    • 
            
  • Pressure parsing does not yet preserve whether the source was QNH or altimeter.
    • 
            
  • Sky and remarks parsing are basic and pattern-dependent.
    • 
            
  • This is a deterministic MVP parser, not a full aviation NLP system.
